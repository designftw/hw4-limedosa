Learnability:
This website was extremely non-trivial to use. It didn't allow for a user-friendly interaction. Customizing any part of it was extremely hard. To better guage its shortcomings, I decided to use it myself. The current template right now is a costumized version of the css, which took a lot of time to acquire. It felt nearly impossible to edit the css in the template.

Possible Upgrades:
One thing that can be changed is allowing users to get a tour of how to use the template. To increase learnability, users should be able to edit the css. This would include giving users a css file and an html file they can edit side by side in a CodePen manner.
